## Story

Como script de pixel rodando no browser,
Quero um módulo publisher central que envie eventos ao worker,
Para que todos os scripts de pixel usem uma única função `track()` com deduplicação automática e geração determinística de `event_id`.

---

## Acceptance Criteria

- [ ] AC1: `worker/src/pixel/core/publisher.ts` exporta função `track(eventName, customData?, eventId?): string`
- [ ] AC2: `track()` retorna o `event_id` gerado ou fornecido
- [ ] AC3: Deduplicação browser-side: mesmo `event_id` na mesma sessão não dispara POST duplicado
- [ ] AC4: Para eventos `Purchase` com `orderId`, `event_id` determinístico: `purchase_{pixelId}_{orderId}`
- [ ] AC5: Para outros eventos, `event_id` é UUID v4
- [ ] AC6: POST para `collect_url` com `keepalive: true` (não bloqueia navegação)
- [ ] AC7: Corpo do POST inclui todos os campos de `NexusEvent` com sinais coletados
- [ ] AC8: Falha silenciosa no POST — `.catch(() => {})` sem throw
- [ ] AC9: `pixelId` lido da config injetada no banner (`__CONFIG__`)

---

## Scope

**IN:**
- Construção do payload `NexusEvent`
- POST para o worker com `keepalive`
- Deduplicação por `event_id`
- `event_id` determinístico para Purchase

**OUT:**
- Disparo de `fbq()`, `ttq()`, `gtag()` (pertence a `platforms/`)
- Lógica de retry (o worker trata falhas de CAPI)

## Dev Notes

### Payload NexusEvent completo

```typescript
function buildPayload(
  nx_user: string,
  signals: SignalMap,
  utms: Record<string, string>,
  eventName: EventName,
  customData?: CustomData,
  eventId?: string,
  config?: PixelConfig,
): NexusEvent {
  const id = eventId ?? generateEventId()
  return {
    event: eventName,
    event_id: id,
    nx_user,
    page_url: window.location.href.split('?')[0],
    page_title: document.title || undefined,
    page_referrer: document.referrer || undefined,
    browser_data: {
      cart_token: readCookie('cart_token') || readCookie('cart') || undefined,
      fbp: signals.fbp,
      fbc: signals.fbc,
      fbclid: signals.fbclid,
      gclid: signals.gclid,
      gbraid: signals.gbraid,
      wbraid: signals.wbraid,
      ttclid: signals.ttclid,
      ttp: signals.ttp,
      msclkid: signals.msclkid,
      ga_client_id: signals.ga_client_id,
    },
    utm_data: Object.keys(utms).length ? utms : undefined,
    custom_data: customData,
    test_event_code: config?.meta_test_event_code,
    tiktok_test_event_code: config?.tiktok_test_event_code,
  }
}
```

### Deduplicação

```typescript
const dedupBuffer = new Set<string>()

function isDuplicate(eventId: string): boolean {
  if (dedupBuffer.has(eventId)) return true
  dedupBuffer.add(eventId)
  return false
}
```

### POST com keepalive

```typescript
function sendEvent(payload: NexusEvent, collectUrl: string): void {
  fetch(collectUrl, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(payload),
    keepalive: true,
  }).catch(() => {})
}
```
