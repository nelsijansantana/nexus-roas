## Story

Como checkout CartPanda,
Quero um script instalado no checkout que capture eventos de funil,
Para que `InitiateCheckout` e `AddPaymentInfo` sejam disparados ao worker e plataformas, com sinais recuperados via URL params (decorados pelo script de origem) e cookies locais.

---

- [ ] AC1: `worker/src/pixel/scripts/cartpanda-checkout.ts` compila para `dist/cartpanda-checkout.js`
- [ ] AC2: Lê `nx_user` de URL params (`src`, `sck`) primeiro, depois cookies `nx_user` e `nx_lid`

- [ ] AC8: Dedup: mesmo evento não disparado mais de uma vez por sessão

## Dev Notes

### Eventos CartPanda no dataLayer

```typescript
// CartPanda emite estes eventos no dataLayer:
const CARTPANDA_EVENT_MAP: Record<string, EventName> = {
  'begin_checkout':       'InitiateCheckout',
  'begin_checkout_info':  'AddShippingInfo',   // variação CartPanda
  'add_payment_info':     'AddPaymentInfo',
}
```

### Leitura de nx_user (prioridade)

```typescript
function resolveNxUser(): string {
  const params = new URLSearchParams(window.location.search)
  // 1. URL params (cross-domain via link decorator)
  const fromUrl = params.get('src') || params.get('sck')
  if (fromUrl) return fromUrl
  // 2. Cookies locais
  const fromCookie = readCookie('nx_user') || readCookie('nx_lid')
  if (fromCookie) return fromCookie
  // 3. localStorage
  try { return localStorage.getItem('nx_user') || '' } catch {}
  return ''
}
```

### Timeout fallback

```typescript
// Se dataLayer não emitir begin_checkout em 2 segundos, forçar InitiateCheckout
setTimeout(() => {
  if (!firedEvents.has('InitiateCheckout')) {
    sendEvent('InitiateCheckout', {})
  }
}, 2000)
```
