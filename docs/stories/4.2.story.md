## Story

Como loja Shopify com checkout nativo,
Quero um script instalado via Customer Events API que capture eventos de checkout,
Para que `InitiateCheckout`, `AddShippingInfo` e `AddPaymentInfo` sejam disparados para o worker e plataformas de tráfego durante o fluxo de checkout sandboxed do Shopify.

## Dev Notes

### Contexto do ambiente sandbox

O Customer Events Sandbox **não tem acesso** a:
- `document.cookie` direto
- `window.localStorage` direto
- DOM da página pai

**Tem acesso via API sandbox:**
- `browser.cookie.get(name)` — lê cookies do checkout domain
- `browser.localStorage.getItem(key)` — lê localStorage do checkout domain
- `analytics` global — para `subscribe()`
- `init` global — chamado na inicialização

### Leitura de sinais no sandbox

```typescript
// No ambiente Customer Events, browser.* são as únicas APIs de storage
const nxUser = browser.cookie.get('nx_user')?.value
  || browser.cookie.get('nx_lid')?.value
  || ''

const fbp = browser.cookie.get('_fbp')?.value
const fbc = browser.cookie.get('_fbc')?.value
const ttp = browser.cookie.get('_ttp')?.value
const fbclid = browser.cookie.get('_nx_fbclid')?.value
```

### Subscribe nos eventos

```typescript
analytics.subscribe('checkout_started', (event) => {
  const checkout = event.data.checkout
  sendEvent('InitiateCheckout', {
    value: checkout.totalPrice?.amount,
    currency: checkout.totalPrice?.currencyCode,
    contents: checkout.lineItems?.map(item => ({
      id: item.variant?.id,
      quantity: item.quantity,
      item_price: item.variant?.price?.amount,
    })),
  })
})

analytics.subscribe('checkout_contact_info_submitted', (event) => {
  const email = event.data.checkout?.email
  sendEvent('AddContactInfo', {}, { email })
})

analytics.subscribe('checkout_shipping_info_submitted', (event) => {
  const addr = event.data.checkout?.shippingAddress
  sendEvent('AddShippingInfo', {}, {
    city: addr?.city,
    state: addr?.provinceCode,
    country: addr?.countryCode,
    zip: addr?.zip,
  })
})

analytics.subscribe('payment_info_submitted', (event) => {
  sendEvent('AddPaymentInfo', {})
})
```

### Placeholders neste script

- `/*__NX_COLLECT__*/` — URL do endpoint `/event`
- `/*__META_PIXEL_IDS__*/` — array de pixel IDs Meta
- `/*__TIKTOK_PIXEL__*/` — pixel ID TikTok
- `/*__GA4_ID__*/` — measurement ID GA4
- `/*__META_TEST__*/` — test event code Meta
- `/*__TIKTOK_TEST__*/` — test event code TikTok
