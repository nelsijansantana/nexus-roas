## Story

Como worker que recebe eventos do browser pixel,
Quero um handler `/event` que processe, atribua e encaminhe cada evento para storage e CAPI,
Para que dados de usuário, sessão e atribuição sejam persistidos no D1 e os eventos enfileirados para dispatch CAPI assíncrono.

---

## Acceptance Criteria

- [ ] AC1: `POST /event` aceita payload `NexusEvent` e retorna `200 { ok: true, event_id }`
- [ ] AC2: Valida `pixel_id` no body — retorna `400` se ausente ou inválido
- [ ] AC3: Extrai geolocalização dos headers Cloudflare: `CF-IPCountry`, `CF-IPCity`, `CF-IPRegion`, `CF-IPPostalCode`
- [ ] AC4: Upsert de `user_store` no D1 com dados do usuário e geo
- [ ] AC5: Insere `user_attribution` com session_id, UTMs, click IDs e canal derivado
- [ ] AC6: Upsert de `checkout_session` quando evento for `InitiateCheckout` ou `AddToCart`
- [ ] AC7: Deduplicação server-side via KV: `dedup:{pixel_id}:{event_id}` TTL 24h
- [ ] AC8: Evento enfileirado em Cloudflare Queue para dispatch CAPI assíncrono
- [ ] AC9: Inserção em ClickHouse (`tracker_events`) de forma assíncrona (não-bloqueante)
- [ ] AC10: `channel` derivado dos UTMs/click IDs (ex: `paid_social_meta`, `organic`, `direct`)

---

## Dev Notes

### Derivação de channel

```typescript
function deriveChannel(signals: SignalMap, utms: Record<string, string>): string {
  if (signals.fbclid) return 'paid_social_meta'
  if (signals.gclid || signals.gbraid || signals.wbraid) {
    return utms.utm_medium?.includes('search') ? 'paid_search_google' : 'paid_display_google'
  }
  if (signals.ttclid) return 'paid_social_tiktok'
  if (signals.msclkid) return 'paid_search_bing'
  if (utms.utm_source) {
    const src = utms.utm_source.toLowerCase()
    const med = (utms.utm_medium || '').toLowerCase()
    if (med === 'organic' || med === 'seo') return `organic_${src}`
    if (med === 'email') return 'email'
    if (med === 'referral') return 'referral'
    return `other_${src}`
  }
  return 'direct'
}
```

### Deduplicação KV

```typescript
const dedupKey = `dedup:${pixel_id}:${event_id}`
const existing = await env.KV_DEDUP.get(dedupKey)
if (existing) return new Response(JSON.stringify({ ok: true, duplicate: true }))
await env.KV_DEDUP.put(dedupKey, '1', { expirationTtl: 86400 })
```
