## Story

Como script de pixel rodando no browser,
Quero um módulo de captura de sinais de marketing,
Para que click IDs, cookies de plataforma e UTMs sejam coletados, persistidos e disponibilizados para todos os eventos.

---

## Acceptance Criteria

- [ ] AC1: `worker/src/pixel/core/signals.ts` exporta função `collectSignals(): SignalMap`
- [ ] AC2: Captura da URL: `fbclid`, `gclid`, `gbraid`, `wbraid`, `ttclid`, `msclkid`, `twclid`
- [ ] AC3: Captura de cookies: `_fbp`, `_fbc`, `_ttp`
- [ ] AC4: Extrai `ga_client_id` do cookie `_ga` (formato `GA1.x.NNNN.NNNN` → `NNNN.NNNN`)
- [ ] AC5: Constrói `_fbc` automaticamente se `fbclid` presente na URL mas cookie `_fbc` ausente (formato: `fb.1.{timestamp}.{fbclid}`)
- [ ] AC6: UTMs persistidos em `localStorage` (`nx_utms`) — novos UTMs sobrescrevem anteriores
- [ ] AC7: `collectUtms(): Record<string, string>` retorna UTMs do `localStorage`
- [ ] AC8: Click IDs salvos em cookies individuais `_nx_{name}` com TTL 30 dias como fallback
- [ ] AC9: Todos os campos opcionais retornam `undefined` (não `null` ou `""`) quando ausentes

### Extração de ga_client_id

```typescript
function extractGa4ClientId(gaCookie: string): string | undefined {
  const parts = gaCookie.split('.')
  if (parts.length >= 4) return `${parts[2]}.${parts[3]}`
  return undefined
}
```

### Persistência de UTMs (localStorage)

```typescript
const NX_UTMS_KEY = 'nx_utms'

function persistUtms(params: URLSearchParams): void {
  const existing = readUtms()
  let updated = false
  UTM_PARAMS.forEach(key => {
    const val = params.get(key)
    if (val) { existing[key] = val; updated = true }
  })
  if (updated) localStorage.setItem(NX_UTMS_KEY, JSON.stringify(existing))
}
```
