## Story

Como worker que recebe webhooks de gateways de pagamento,
Quero handlers dedicados para Shopify, CartPanda, Yampi e genérico,
Para que cada webhook seja validado via HMAC, deduplicado, persistido como raw e enfileirado para processamento CAPI assíncrono.

---

## Acceptance Criteria

- [ ] AC1: `POST /gateways/shopify` com HMAC validation via `x-shopify-hmac-sha256`
- [ ] AC2: `POST /gateways/cartpanda` com HMAC validation via header CartPanda
- [ ] AC3: `POST /gateways/yampi` com HMAC validation via `x-yampi-hmac-sha256`
- [ ] AC4: `POST /gateways/:gateway` handler genérico para demais 11 gateways
- [ ] AC5: Deduplicação via KV: `wh:{pixelId}:{gateway}:{orderId}` TTL 24h
- [ ] AC6: `insertWebhookRaw(d1, {...})` persiste payload bruto + metadados
- [ ] AC7: Evento enfileirado em Cloudflare Queue após dedup e persist
- [ ] AC8: Retorna `200 { ok: true }` imediatamente (não aguarda processamento)
- [ ] AC9: Retorna `401` para HMAC inválido
- [ ] AC10: `pixel_id` resolvido do path, query param, ou header configurado

## Dev Notes

### Deduplicação webhook

```typescript
const dedupKey = `wh:${pixelId}:${gateway}:${orderId}`
const exists = await env.KV.get(dedupKey)
if (exists) return new Response(JSON.stringify({ ok: true, duplicate: true }), { status: 200 })
await env.KV.put(dedupKey, '1', { expirationTtl: 86400 }) // 24h
```

### insertWebhookRaw

```typescript
`).bind(pixelId, gateway, orderId, JSON.stringify(body), Date.now()).run()
```

### Resolução de pixel_id

```typescript
function resolvePixelId(request: Request, params: Record<string, string>): string | null {
  // 1. Path param: /gateways/shopify/:pixel_id
  if (params.pixel_id) return params.pixel_id
  // 2. Query param: ?pixel_id=XXX
  const url = new URL(request.url)
  const qp = url.searchParams.get('pixel_id')
  if (qp) return qp
  // 3. Header customizado
  const hdr = request.headers.get('X-Pixel-Id')
  if (hdr) return hdr
  return null
}
```
