## Story

Como consumer do Cloudflare Queue,
Quero um processor que faça identity matching e despache para todos os CAPIs configurados,
Para que cada webhook de gateway resolva o `nx_user` e sinais de tracking via múltiplos fallbacks antes de enviar para Meta, TikTok, GA4 e Google Ads.

---

## Acceptance Criteria

- [ ] AC1: `worker/src/queue/processor.ts` exporta `processGatewayEvent(msg, env): Promise<void>`
- [ ] AC2: Identity resolution em 4 níveis (por prioridade):
  1. `nx_user` do próprio payload do gateway (metadata, note_attributes)
  2. `cart_token` → lookup em `checkout_sessions`
  3. `email + phone` → lookup em `user_store`
  4. `no_match` — prossegue sem sinais históricos
- [ ] AC3: Attribution recovery: após resolver nx_user, busca `user_attribution` mais recente (últimos 30 dias)
- [ ] AC4: Para Yampi+Shopify: `_nx_user` de `metadata.data` tem prioridade sobre matching por email
- [ ] AC5: Dispatch para: Meta CAPI, TikTok CAPI, GA4, Google Ads — todos assíncronos em `Promise.allSettled()`
- [ ] AC6: Resultado de cada CAPI atualizado em `webhook_raw`: `capi_meta`, `capi_tiktok`, `capi_ga4`
- [ ] AC7: `match_type` registrado no ClickHouse: `nx_user` | `cart_token` | `email_phone` | `no_match`
- [ ] AC8: Erros de CAPI não bloqueiam os demais (isolamento por `Promise.allSettled`)
- [ ] AC9: Consumer registrado no `worker/src/queue/consumer.ts`

## Dev Notes

### Identity Resolution Flow

```typescript
async function resolveIdentity(event: ParsedGatewayEvent, env: Env): Promise<IdentityResult> {
  // Nível 1: nx_user direto do payload
  if (event.nx_user) {
    const attribution = await recoverAttribution(event.nx_user, event.pixel_id, env)
    return { nx_user: event.nx_user, match_type: 'nx_user', attribution }
  }

  // Nível 2: cart_token → checkout_sessions
  if (event.cart_token) {
    const session = await env.D1.prepare(
      'SELECT nx_user, fbclid, gclid, ttclid, fbp, fbc, ttp FROM checkout_sessions WHERE token = ? AND pixel_id = ?'
    ).bind(event.cart_token, event.pixel_id).first<CheckoutSession>()

    if (session?.nx_user) {
      return { nx_user: session.nx_user, match_type: 'cart_token', signals: session }
    }
  }

  // Nível 3: email + phone → user_store
  if (event.customer.email || event.customer.phone) {
    const user = await env.D1.prepare(`
      SELECT nx_user FROM user_store
      WHERE pixel_id = ? AND (email = ? OR phone = ?)
      LIMIT 1
    `).bind(event.pixel_id, event.customer.email, event.customer.phone).first<UserStore>()

    if (user?.nx_user) {
      const attribution = await recoverAttribution(user.nx_user, event.pixel_id, env)
      return { nx_user: user.nx_user, match_type: 'email_phone', attribution }
    }
  }

  // Nível 4: no_match
  return { nx_user: null, match_type: 'no_match', attribution: null }
}
```

### Attribution Recovery

```typescript
async function recoverAttribution(nxUser: string, pixelId: string, env: Env): Promise<AttributionData | null> {
  return env.D1.prepare(`
    SELECT fbclid, fbc, fbp, gclid, gbraid, wbraid, ttclid, ttp, ga_client_id
    FROM user_attribution
    WHERE nx_user = ? AND pixel_id = ?
    ORDER BY created_at DESC
    LIMIT 1
  `).bind(nxUser, pixelId).first<AttributionData>()
}
```
