## Story

Como loja Shopify,
Quero um script instalado no tema que capture eventos de storefront e sincronize sinais com o carrinho,
Para que `nx_user`, click IDs e UTMs sejam transferidos automaticamente para o checkout Shopify nativo, CartPanda ou Yampi via `cart.attributes`.

## Dev Notes

### Sincronização cart.attributes

```typescript
async function syncCartAttributes(nxUser: string, signals: SignalMap, utms: Record<string, string>): Promise<void> {
  const attrs: Record<string, string> = { nx_user: nxUser }

  // UTMs
  const utmKeys = ['utm_source', 'utm_medium', 'utm_campaign', 'utm_content', 'utm_term', 'utm_id', 'src', 'sck']
  utmKeys.forEach(k => { if (utms[k]) attrs[k] = utms[k] })

  // Click IDs e cookies de plataforma
  const signalKeys = ['fbclid', 'fbc', 'fbp', 'gclid', 'gbraid', 'wbraid', 'ttclid', 'ttp', 'msclkid'] as const
  signalKeys.forEach(k => { if (signals[k]) attrs[k] = signals[k]! })

  await fetch('/cart/update.js', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ attributes: attrs }),
  }).catch(() => {})
}
```

### Detecção de ambiente Shopify

O script verifica `window.Shopify` para ativar funcionalidades específicas:

```typescript
const isShopify = typeof window !== 'undefined' && !!(window as any).Shopify
if (isShopify) {
  syncCartAttributes(nxUser, signals, utms)
  document.addEventListener('cart:updated', () => syncCartAttributes(nxUser, signals, utms))
}
```

### Contexto de injeção no tema

Instalação no `theme.liquid`:
```html
<script src="{{ shop.url }}/apps/nexus/shopify-storefront.js?pixel_id={{ settings.nexus_pixel_id }}"></script>
```

Ou servido pelo worker como:
```
GET /scripts/shopify-storefront.js?pixel_id=XXXX
```

### Placeholders utilizados

- `/*__CONFIG__*/` — substituído por `const __CONFIG__ = { collect_url: "...", meta_pixel_id: "...", ... }`
- `/*__NX_USER__*/` — substituído por `const __NX_USER__ = "nxu_..."` (injeção server-side opcional)
